## Overview

CppDrg is a modding framework for Deep Rock Galactic written in C++. It provides:

- **Game Hooking System** - Intercept and modify game function calls in real-time
- **Command Handler** - Execute custom commands through the mod system
- **Property Inspector** - Inspect and modify game object properties at runtime
- **Variable System** - Dynamic variable management for configuration and state
- **Safe String Library** - Utility functions for string manipulation
- **Logging Framework** - Comprehensive logging using spdlog

The framework is compiled as a DLL that injects into the game process and provides modding capabilities without requiring source code modifications to Deep Rock Galactic.

## Dependencies

### External Libraries

- **spdlog** - Fast C++ logging library (included via vcpkg)
  - Used for comprehensive logging throughout the framework
  - Header-only or compiled version
  - [GitHub](https://github.com/gabime/spdlog)

- **MinHook** - Hooking library wrapped by EasyHook
  - Provides the underlying hooking mechanism for game function interception
  - Used in `Lib_EasyHook.h` for setting up API hooks
  - EasyHook is a C++ wrapper around MinHook for easier integration
  - [GitHub](https://github.com/TsudaKageyu/minhook)
